Command Line Tools

curl, wget, git, npm

  • curl:
    curl -x http://127.0.0.1:3128 https://example.com
  • wget:
    wget -e use_proxy=yes -e http_proxy=127.0.0.1:3128 https://example.com
  • git:
    git config --global http.proxy http://127.0.0.1:3128

Environment Variables

Universal proxy configuration

  • Bash/Zsh: Add to ~/.bashrc or ~/.zshrc
    export HTTP_PROXY=http://127.0.0.1:3128 export HTTPS_PROXY=http://127.0.0.1:3128 export NO_PROXY=localhost,127.0.0.1,.local
  • Windows PowerShell:
    $env:HTTP_PROXY = "http://127.0.0.1:3128" $env:HTTPS_PROXY = "http://127.0.0.1:3128"
  • SOCKS5 Proxy:
    export ALL_PROXY=socks5://127.0.0.1:1081
